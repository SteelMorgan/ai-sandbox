Это “скелет проекта”, который можно автоматически заливать в новый репозиторий через `scripts/provision-repo.ps1 -SeedPath ...`.

Сюда клади то, что ты хочешь иметь **в каждом проекте по умолчанию**:
- `.cursor/skills/...` и `.cursor/rules/...` (проектные, которые должны ехать вместе с репо)
- `docs/` (регламенты, договорённости)
- базовые конфиги (lint/format/CI), если нужно

Важно:
- “глобальные” skills/rules (host-level) сюда смысла класть нет — они живут в `~/.cursor/...` на машине.
